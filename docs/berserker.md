# Select debuff

On **Jump 4**, Berserker intentionally gets hit by **both Ice and Fire**. This allows to use Rage with deadly blow, when a phantom Oracle provides invulnerability. See [Deadly Blowmaxxing](https://docs.google.com/spreadsheets/d/1_w3qAdhK1dkgA3TV5g58ImQbbiWsh547Lyje0sz9tCQ/edit?gid=2082321912#gid=2082321912).
